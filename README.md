# Priority Sorter

A drag-and-drop prioritization app for reordering items across paginated lists.

Built using basic Vite template with React 19.2. Tested with Node.js 22+

## Features

- **Drag & drop reordering** within the current page
- **Pagination** for large lists (10 to 30 items per page)
- **Quick actions** — drop zones to move items to previous, next, first or last page
- **Export final order** — print the complete prioritized list with numbering

## Setup

```bash
# Install dependencies and run
npm install
npm run dev
```
